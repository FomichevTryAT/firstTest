Мой проект с первыми автотестами сайта заказа самокатов https://qa-scooter.praktikum-services.ru/

Класс DoOrder проверяет функционал заказа самоката.  
Класс GoToScooter проверяет переход на главную страницу при нажатии на логотип самоката.  
Класс GoToYandex проверяет переход в поиск Яндекса при нажатии на логотип Яндекса.  
Класс ImportantQuestions проверяет открытие и содержимое Вопросов о важном внизу Сайта.  
Класс CheckOrder проверяет существует ли заказ по номеру.  

PageObject:  
MainPage - локаторы главной страницы.  
OrderPage - локаторы страницы при нажатии кнопки Заказа.  
SecondOrderPage - продолжение заказа.
